# 화면크기 조절
- alt + 마우스 휠 위아래로 조절
- ctrl + 0 > 화면크기에 맞춤
- ctrl + 1 > 100%크기로 조절

# 레이어 복사
- ctrl + j
- alt + 레이어를 위로 이동 > 복사된 레이어가 위에 위치
- alt + 레이어를 아래로 이동 > 복사된 레이어가 아래에 위치

# 이미지 크기 조절
- ctrl + T
- 조절시 alt + 휠 조절 > 기준을 가운데로

# group 만들기
